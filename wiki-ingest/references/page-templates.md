# Page body templates

Frontmatter is generated by `scripts/new_page.py` and its schema is owned by the repo's
`CLAUDE.md` — this file only covers the **body** below the frontmatter. Section headers
are a default, not a straitjacket: drop a section that would be empty, but keep the
order for the ones you use so pages stay scannable.

## Source page (`wiki/sources/`)

```markdown
# <Title>

**Raw file:** `raw/<filename>` · **Type:** <article | paper | notes | transcript | ...>
**Author/origin:** <if known> · **Published:** <if known>

## Summary
3–8 sentences. What this source is and its central argument or content.

## Key claims
- Claim one, stated faithfully to the source — even if you doubt it.
- Claim two. Link entities/concepts inline: [[entities/foo]], [[concepts/bar]].

## Notable specifics
Numbers, dates, definitions, or short quotes (keep quotes minimal) that entity/concept
pages will want to cite precisely.

## Affects
- [[entities/...]] — what this source adds to that page
- [[concepts/...]] — same
```

## Entity page (`wiki/entities/`)

```markdown
# <Name>

One-sentence identification (who/what this is).

## Overview
Narrative synthesis across all sources. Every claim cites: (source: [[sources/<slug>]]).

## Relationships
- [[entities/other]] — nature of the relationship (source: ...)

## Timeline
- YYYY-MM — event (source: ...)   ← only if the entity has meaningful chronology
```

## Concept page (`wiki/concepts/`)

```markdown
# <Concept>

One-sentence definition.

## Explanation
What it is, why it matters, how sources characterize it. Cite per claim.

## Positions and disagreements
Where sources differ on this concept — this section often hosts ⚠️ Conflict notes.

## See also
- [[concepts/related]] — how it relates
```

## Conflict note format (any page)

Place immediately after the claim it concerns:

```markdown
> ⚠️ Conflict: [[sources/2026-07-01-a]] says X; [[sources/2026-07-03-b]] says Y.
> Not adjudicated — flagged for review.
```

If adjudicated, replace the second line with the reason ("B is newer and explicitly
corrects A"), and set the losing claim's phrasing to past-tense attribution rather
than deleting it.
